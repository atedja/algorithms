### Problem

Given an integer `amount` and an array of `denoms` representing coin values, return the change with the least amount of coins.

### Example

    amount: 42
    demons: [25, 10, 5, 1]
    return: [25, 10, 5, 1, 1]
    
    amount: 50
    demons: [25, 10, 5, 1]
    return: [25, 25]
    
    amount: 16
    demons: [10, 8, 1]
    return: [8, 8]
